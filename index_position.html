Positioning elements with HTML5

In HTML5 you have 4 different positioning values you can assign to elements.
These determine where the element will be placed in the flow of the page.

The Static position

The static positioned element is not affected by top, left, right and bottom properties.
It is not positioned in any special way; it always follows the normal flow of the page.

I am a static div. I follow the normal flow of this page, that is why I am positioned underneath the above text.

If you inspect this page in Google Chrome, you can see the static ID has top: 100px.
It has no effect on this element. Try changing the value and see for yourself.
#static { position: static; border: 2px solid rgb(10,15,200); width: auto; height: auto; margin: auto; top: 100px; }

The Fixed position

The fixed positioned element is relative to the viewport, meaning it will always stay in the same place.
This also means if you scroll the page up and down, it will stay where you put it.
For this element you can use the top, bottom, left and right properties to position it.

I am a fixed div. I stay in place, no matter if you scroll or change the viewport size.

#fixed { position: fixed; border: 2px solid rgb(200,15,40); width: auto; height: auto; margin: auto; top: 500px; left: 50px; }

The Absolute position

An absolute positioned element is taken out of the normal page flow. Its position is relative to its parent. If it has no clear parent, it will look at the html/body tag and position itself accordingly.
You can position an absolute element with the top, bottom, left and right properties.

This is an absolute positioned element. Absolute positions means the element is taken out of the normal page flow. #absolute { position: absolute; border: 2px solid rgb(25,200,25); width: auto; height: auto; margin: auto; top: 1200px; left: 155px; }
I am a relative positioned element, holding an absolute child element. #relative-parent { position: relative; border: 2px solid rgb(200,100,100); width: auto; height: 200px; margin: auto; }
I am absolute. I look at the position of my parent. If my position is 0,0,0,0, then I place myself in the top left corner of my parent container.
My top is now 25px, so that you can read this. #absolute-child { position: absolute; border: 2px solid rgb(0, 200, 50); top: 25px; left: 0; bottom: 0; right: 0; }

The Relative position

Relative elements position themselves relative to their normal position. They follow the flow of the page.

I am relative. I'm a bit like a static positioned element, but you can adjust me with the top, left, bottom, right properties. #relative { position: relative; border: 2px solid rgb(150,150,0); height: auto; width: auto; }
